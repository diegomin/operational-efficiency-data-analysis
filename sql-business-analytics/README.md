# SQL Business Analytics Portfolio

Este repositorio contiene análisis SQL orientados a negocio,
simulando escenarios reales de una empresa retail.

## Objetivo
Demostrar habilidades de SQL aplicadas a:
- Ventas
- Operaciones
- Inventarios

Cada query está escrita y explicada como lo haría un analista
en un entorno corporativo.

## Tecnologías
- SQL (MySQL / PostgreSQL compatible)
- Enfoque analítico y de negocio
